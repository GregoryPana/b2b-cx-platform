# Governance Dashboard - Installation Assessment Platform User Guide

This guide explains how to use the Installation Assessment platform inside the Governance Dashboard.

---

## 1) Accessing Installation Governance

1. Open the Governance Dashboard link.
2. If a security warning appears, click **Advanced**.
3. Click **Proceed to site** (or **Continue to website**).
4. Sign in with your work account.
5. Select **Installation Assessment** from platform selection.

### Access Required

- Installation Admin
- Super Admin

### Success Check

You should see:

- Platform overview section
- Planned modules section

Tabs in this platform:

- This Governance installation module currently does not show operational tabs.
- It shows informational sections only (overview and planned modules).

## 2) Current Functionality

This platform is currently informational in Governance.

### What you can do

- View platform structure.
- Understand planned capabilities.

### What you cannot do

- No analytics tab in this module view.
- No review queue actions.
- No approvals from this screen.
- No survey execution from this Governance page.

## 3) Common Situations

- **You cannot find analytics/review tabs:** This module is informational only in Governance.
- **Access denied:** Confirm Installation Admin or Super Admin role.
- **Page does not load correctly:** Refresh and sign in again, then reselect Installation Assessment.
