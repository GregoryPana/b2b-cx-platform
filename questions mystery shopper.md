# CWS Mystery Shopper Assessment – Question Framework

This document defines the full question structure for the **Mystery Shopper Assessment Form**, including:

- Categories
- Questions
- Standardized input types

All qualitative rating questions use the **standardized 5-point scoring model**:

| Score | Label             |
| ----- | ----------------- |
| 5     | Excellent         |
| 4     | Good              |
| 3     | Fair / Acceptable |
| 2     | Poor              |
| 1     | Very Poor         |

---

# SECTION 1 — Visit Details

Purpose: Capture metadata about the mystery shopper visit.

| Question                        | Input Type                                                                                  |
| ------------------------------- | ------------------------------------------------------------------------------------------- |
| Customer Service Centre Visited | Dropdown (Location List)                                                                    |
| Date of Visit                   | Date                                                                                        |
| Time of Visit                   | Time                                                                                        |
| Purpose of Visit                | Dropdown (Service Category - General Enquiry, Billing, Device, Broadband, Complaint, Other) |
| Name of Staff on Duty           | Text                                                                                        |

---

# SECTION 2 — External Environment & First Impression

Purpose: Evaluate the first impression of the store and greeting behaviour.

| Question                                  | Input Type   |
| ----------------------------------------- | ------------ |
| How is Signage visibility & cleanliness   | Score (1–5) |
| How is Store entrance cleanliness         | Score (1–5) |
| Rate the display of Promotional materials | Score (1–5) |
| Greeted within 30 seconds                 | Yes / No     |
| Greeting polite & warm                    | Yes / No     |

---

# SECTION 3 — Staff Appearance & Professionalism

Purpose: Evaluate staff appearance and professionalism.

| Question                                 | Input Type   |
| ---------------------------------------- | ------------ |
| Are staff Uniform neat and complete      | Score (1–5) |
| How is the staff Personal grooming of    | Score (1–5) |
| Rate the Name badge visibility           | Score (1–5) |
| Do staff act with Professional behaviour | Score (1–5) |

---

# SECTION 4 — Customer Service Interaction

Purpose: Evaluate quality of customer interaction and service delivery.

| Question                                     | Input Type   |
| -------------------------------------------- | ------------ |
| Did staff listen actively                   | Yes / No     |
| Did staff Ask relevant questions            | Yes / No     |
| Rate Staff Product/service knowledge         | Score (1–5) |
| How was the Accuracy of information Provided | Score (1–5) |
| Handled clarifying questions                 | Score (1–5) |
| Was a Solution provided                     | Yes / No     |
| Was the Solution helpful                     | Yes / No     |

---

# SECTION 5 — Store Environment & Comfort

Purpose: Evaluate the physical store environment and customer comfort.

| Question                         | Input Type   |
| -------------------------------- | ------------ |
| Cleanliness of store             | Score (1–5) |
| Queue management                 | Score (1–5) |
| Display organisation and posters | Score (1–5) |
| Air conditioning / ventilation   | Score (1–5) |
| Space & comfort                  | Score (1–5) |

---

# SECTION 6 — Time & Efficiency

Purpose: Evaluate operational efficiency and service speed.

| Question                           | Input Type                                             |
| ---------------------------------- | ------------------------------------------------------ |
| Waiting time                       | Multiple Choice (Under 3 / 3–7 / 7–15 / 15+ minutes) |
| Service completion (handling time) | Multiple Choice (Quick / Acceptable / Slow)            |

---

# SECTION 7 — Overall Experience (CSAT & NPS)

Purpose: Measure satisfaction and likelihood to recommend.

| Question                         | Input Type    |
| -------------------------------- | ------------- |
| Staff interaction satisfaction   | Score (0–10) |
| Store environment satisfaction   | Score (0–10) |
| NPS – Recommend this outlet     | NPS (0–10)   |
| Final comments / recommendations | Text          |

---

# SECTION 8 — Mystery Shopper Info (Internal)

Purpose: Internal tracking and audit trail.

| Question              | Input Type          |
| --------------------- | ------------------- |
| Shopper Name          | Text                |
| Date Completed report | Auto-generated Date |

---

# Input Type Summary

| Input Type      | Usage                            |
| --------------- | -------------------------------- |
| Score (1–5)    | Service quality evaluation       |
| Yes / No        | Behaviour compliance checks      |
| Dropdown        | Structured categorical selection |
| Multiple Choice | Operational performance          |
| Score (0–10)   | Satisfaction rating              |
| NPS (0–10)     | Loyalty measurement              |
| Text            | Qualitative feedback             |
| Date / Time     | Visit tracking                   |
